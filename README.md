# Taak_infm_lander_van_avermaet
Taak vak infmormatiemanagement Lander Van Avermaet 2025-2026

Om met MQTTX te verbinden met de Mosquitto-broker:

Open MQTTX.
Maak een nieuwe verbinding aan met de volgende instellingen:

Naam: Lokale Mosquitto
Host: localhost
Poort: 1883
Client ID: (mag leeg zijn of een willekeurige naam)

Klik op Connect.
Je kunt nu berichten publiceren en ontvangen
